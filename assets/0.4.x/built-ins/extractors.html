Extractors#

The following extractors are already built in and you can use them by turning on responsive features.

FeatureExtractorDescription
HeaderExtract header information
cookieCookiesExtract cookies information
limitesLimitsExtract limits settings
formFormExtract from-data data
jsonJsonExtract JSON data
multipartMultipartExtract multipart data
paramsParamsExtract parameters from the request
queryQueryExtract query string from the request
sessionSessionExtract session
stateStateExtract state data
websocketWebsocketExtract WebSocket data
