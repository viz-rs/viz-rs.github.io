提取器#

以下是已经内置的提取器，开启响应特性就可以使用它们。

特性提取器说明
Header提取 header 信息
cookieCookies提取 cookies 信息
limitesLimits提取 limits 设置
formForm提取 from-data 数据
jsonJson提取 JSON 数据
multipartMultipart提取 multipart
paramsParams从请求路径中提取参数 params
queryQuery从请求 query string 提取参数
sessionSession提取会话信息
stateState提取 state 状态数据
websocketWebsocket提取 WebSocket 连接
