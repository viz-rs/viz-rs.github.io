## Development

1. Install [`cargo-make`], [`trunk`]

```bash
$ cargo install --force cargo-make trunk
```

2. Serve `en`, `zh-CN`, `zh-TW` or other languages

```bash
$ cargo make dev
```

3. Build

```bash
$ cargo make build
```
